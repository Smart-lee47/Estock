About Stockish

A front-end warehouse management system built with HTML, CSS, and JavaScript — designed to demonstrate real-world inventory and storage concepts.

What is Stockish?

Stockish is a simulated warehouse management company website. It allows businesses to register, choose a warehouse, select a service, and manage their inventory — all through a clean, interactive front-end interface.

How It Works

Companies register through the signup page. Their data is saved using browser localStorage — no backend or database required. The dashboard lets them add and remove products in real time, while the admin panel shows a full report of all registered companies and stored inventory.

The 5 Warehouses

Stockish operates 5 purpose-built warehouses — General Storage, Cold Storage, Bulk Storage, Secure Storage, and a Logistics Hub — each designed for a specific category of goods and business need.

Technology Used

This project was built entirely using front-end web technologies:

HTML5 CSS3 JavaScript localStorage DOM Manipulation

Key Features

Company registration with data persistence, a live inventory dashboard, real-time stock counters, product add and delete functionality, and a full admin report panel — all without a single line of backend code.

Project Purpose

Stockish was developed as a TVET end-of-year project to demonstrate skills in HTML structure, CSS styling, JavaScript logic, DOM manipulation, and data management using browser localStorage.
